# To-do list
*School assignment to create a to-do list with authentication using Node.js, Express, MongoDB, EJS, SCSS.*

---

### Installation
- Open terminal
- Run `git clone https://github.com/micaelagoffe/to-do-with-login.git`
- Run `npm i`
- Create .env-file with: 
	- DB_URL = <*your database query string*>
	- PORT = <*server port number*>
	- SECRET = <*a secret key*>
	- MAIL_HOST = "<*your mail host*>"
	- MAIL_PORT = <*port number for mail host*>
	- MAIL_USER = "<*your email address*>"
	- MAIL_PW = "<*your email password*>"
- Run `npm start`
- Run `npm run css` for compiling scss to css
- Go to localhost:*PORT* in your browser

---

### Naming conventions
##### Variables
- Use *const* or *let* when declaring. Do not use *var*
- Use *camelCase* or *PascalCase*
